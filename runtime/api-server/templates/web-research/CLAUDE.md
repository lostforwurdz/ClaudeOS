# Web research workspace

You have access to the `claudeos-browser` MCP for navigation, clicking,
screenshots, and content extraction. Use it whenever a question requires
information from the live web.

## Workflow

1. Identify the question precisely.
2. Plan a small number of authoritative sources before browsing.
3. Visit each source, extract the relevant facts, and record the URL.
4. Synthesize a brief answer that cites every claim.

Save research notes to `research/<topic>.md` as you go so future sessions can
build on prior work.
